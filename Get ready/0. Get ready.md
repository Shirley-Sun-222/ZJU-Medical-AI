# Pyhton工作环境安装流程

## 软件包管理conda安装

> 注意：本教程仅针对Windows电脑，对于其他系统，请自行学习怎么安装

使用 Anaconda 可以轻松配置和隔离不同项目环境，避免包冲突，让安装和运行常用科学计算库更加简单快捷，也方便课程统一环境和复现，不建议直接使用系统自带 Python。

先下载并安装anaconda：https://www.anaconda.com/download/success (建议安装miniconda，文件小一些，功能一样，只不过没有一些基础包，课程中也用不上)

安装后打开anaconda prompt（请在安装后，在开始栏里面搜索 Anaconda Prompt），后面安装包的操作请在命令行里面进行。

你也应在Windows terminal里面可以看见command前面出现 (base) 这样的标识，说明conda安装成功，如果只在anaconda prompt里面有，但Windows terminal里面没有，请参考以下进行设置：

```bash
# initialize conda for PowerShell
conda init powershell

# or initialize conda for cmd
conda init cmd.exe
```

![image](./assets/image-20250222223156-2cwojfq.png)

安装成功是这个样子，命令行前出现base。

常用conda环境命令如下：

```bash
# 创建环境（指定python版本）
conda create --name myenv python=3.8

# 激活环境
conda activate myenv

# 查看所有环境
conda env list

# 安装包
conda install package_name

# 查看已安装包
conda list

# 删除环境
conda remove --name myenv --all

# 退出环境
conda deactivate
```

## 课程必要软件包安装

### 整体安装

在终端（命令提示符或 PowerShell）中执行：

> 说明：本课程统一创建的环境名称为 `aidrug`​，这样命名便于后续管理和识别相关环境。  
> 建议在 `aidrug` 环境下操作，确保其它项目环境不受影响。

```bash
conda create --name aidrug python=3.11
conda activate aidrug
```

在已激活的 `aidrug` 环境中执行：

```bash
pip install -r requirements.txt
```

或逐条安装：

```bash
pip install rdkit numpy pandas matplotlib scikit-learn tqdm openpyxl openai scipy pyDOE jupyterlab ipywidgets jupyterlab_widgets lckr_jupyterlab_variableinspector torch==2.6.0 pymodbus[serial] opencv-python torchvision
```

如果某个依赖包安装异常或出现冲突，可先卸载再重新安装。例如以 `rdkit` 为例：

```bash
pip uninstall rdkit
pip install rdkit
```

### 删除并重装环境

如果遇到环境损坏或想彻底重装环境，可以先删除整个 `aidrug` 环境，再重新创建和安装：

```bash
# 先关闭已经激活的环境（如有），可以直接关闭终端
# 删除 aidrug 环境
conda remove --name aidrug --all
```

## 编程界面IDE

推荐使用 Cursor 编辑器进行编程与调试，其次可以使用 PyCharm。

Cursor：https://www.cursor.so/  （基于 VS Code，集成 AI 助手，强烈推荐，这是未来的编辑器，建议淘宝买账号使用，如果实在不愿意付钱的，可以考虑使用Kimi的国产替代版本，个人觉得效果差不少）

PyCharm：https://www.jetbrains.com/pycharm/  （功能强大，代码补全、调试体验优秀）

VS Code：https://code.visualstudio.com/  （微软出品，免费，插件生态丰富，可编辑和运行 .ipynb）

JupyterLab：https://jupyter.org/  （交互式笔记本，强烈推荐新手使用，支持代码和可视化。本课程将在课堂演示中使用 JupyterLab，建议大家提前熟悉其操作。）

如何打开 JupyterLab：

1. 确保已激活 `aidrug` conda 环境（或你创建的环境）。
2. 在终端输入以下命令启动 JupyterLab：

```bash
jupyter lab
```

3. 浏览器会自动打开 JupyterLab 页面。如果没有自动打开，可以在浏览器手动访问终端显示的本地地址（如 http://localhost:8888）。

### 在 Cursor 中选择 conda (aidrug) 环境

1. 打开 Cursor，按 <kbd>Ctrl</kbd>​+<kbd>Shift</kbd>​+<kbd>P</kbd>​，输入并选择 `Python: Select Interpreter`。
2. 找到带有 `aidrug` 的 conda 环境，点击选择即可。

如果没有看到 conda 环境，可以重启 Cursor 或确保已安装 Python 插件。
